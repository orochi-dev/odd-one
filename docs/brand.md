# Odd One brand

## Idea

Odd One is a midnight game show with nobody behind the curtain. It should feel theatrical, quick, slightly suspicious, and unmistakably playful without looking like a casino.

The mark is a ring containing three paired signals and one isolated fluorescent signal. The voice is concise: challenge the player, explain irreversible actions plainly, and never use investment, gambling, prize, or speculative language.

## Color

| Token | Value | Use |
| --- | --- | --- |
| Void | `#09090F` | Main background |
| Stage | `#15131F` | Elevated panels |
| Electric lime | `#D8FF3E` | Primary action, winner, focus |
| Ultraviolet | `#7657FF` | Hidden state and identity |
| Signal orange | `#FF5C35` | Reveal phase and warnings |
| Ice | `#E8F5FF` | Cool high-contrast detail |
| Chalk | `#F6F4EE` | Primary text and light surfaces |

Lime is never the sole winner indicator: winner states also use a trophy, number, and explicit label. Orange warnings always contain readable text.

## Type and spacing

- Unbounded Variable: wordmark, headlines, numbers
- Space Grotesk Variable: UI and body
- IBM Plex Mono: timers, room IDs, wallet fragments, technical labels
- Base spacing unit: 4px. Primary rhythm: 8, 12, 20, 28, 40, 60, 80px.

## Motion

Use movement to show phase changes, not decorate every action. The reveal transitions concealed marks to numbers; settlement turns the unique number lime. `prefers-reduced-motion` removes translations, repeated pulses, and rotations.

## Logo

Maintain clear space equal to one signal dot. Never recolor the isolated signal red, add token symbols, put the mark inside a coin, or stretch the ring. Monochrome variants are reserved for small or single-ink contexts.

## Accessibility

Text meets WCAG AA contrast. Focus uses a 3px lime outline with offset. Every phase has an icon, label, and heading. Timers update visually without stealing screen-reader focus.
