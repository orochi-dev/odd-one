# Launch checklist

- Run `npm ci` on a clean checkout.
- Run `npm run verify:full` and archive the evidence.
- Reconfirm current Talent, Celo, MiniPay, and Stacks program requirements.
- Initialize Git only when ready and publish a public repository for Stacks contribution evaluation.
- Deploy and test both contracts on testnets.
- Verify the Celo contract source.
- Test MiniPay automatic wallet access on a physical supported device.
- Deploy both contracts to the intended mainnets.
- Update public contract IDs and deployment blocks.
- Deploy the production web application with its canonical URL.
- Re-run production smoke, responsive, wallet, accessibility, and explorer checks.
- Confirm that no reveal ticket, mnemonic, private key, or deployment secret is present in the repository or build output.
- Submit the deployed project through the account-owned Talent flow.

Code-ready delivery alone is not a contest submission and does not claim eligibility.
